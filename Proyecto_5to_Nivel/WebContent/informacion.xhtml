Información General de la Aplicación

Esta aplicación web ha sido diseñada para recopilar los datos de los nacimientos registrados en el Registro Civil Ecuatoriano, mismas que serán procesadas en una agradable interfaz gráfica cómoda para el usuario

Se podrá elegir a gusto los tipos de datos específicos que se necesitará ver, mismos que se mostrarán con datos tanto cuantitativos, como a nivel gráfico para una mejor comprensión.

Ofrece diversos reportes y gráficos estadísticos Mensuales y Anuales de los datos que se deseen analizar de una forma muy dinámica e intuitiva.
